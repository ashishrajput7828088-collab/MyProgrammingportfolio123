C++ Mini Projects

This repository contains several beginner-friendly C++ programs that demonstrate basic programming concepts such as loops, conditions, functions, and object-oriented programming.

 Project List
1.  Calculator

A simple calculator that performs basic arithmetic operations:

Addition

Subtraction

Multiplication

Division

2.  Number Guessing Game

A fun game where the computer generates a random number and the player has to guess it.
Features:

Random number generation

Attempts counter

Feedback: Too high / Too low

3.  ATM Management System

A simple ATM simulator with these functionalities:

Check Balance

Deposit Amount

Withdraw Amount

Secure PIN system

4. Grade Calculator

A program that calculates a student’s grade based on marks input.
Features:

Calculates average

Assigns grade (A, B, C, D, F)

Validates input

How to Run

Clone the repository:

git clone https://github.com/your-username/your-repository.git


Navigate into the project folder

cd your-repository


Compile a program using g++

g++ calculator.cpp -o calculator
./calculator


(Replace calculator.cpp with any other file name.)
